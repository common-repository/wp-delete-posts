== Description ==

Delete unwanted posts by keyword

Get rid of unwanted blogposts on your blog! How embarrassing, eh?!

== Installation ==

1. Download the .zip file
2. Extract the zip-file, containing the folder "wp-delete-posts"
3. Upload to your wp-content/plugins/ folder on your blog
4. Log in to your blog, go to the plugins page and look for "WP Delete Posts" and activate the plugin
5. You can now access the plugin in your "Tool" menu under "WP Delete Posts"

== Frequently Asked Questions ==

= What happens to my posts? =

All of them will be completely erased from the database. 
There is no recovery option if you check to purge them from trash. 
Remember to back up your database first in case a mistake happens!
